## How to Download and Use

1. **Download Images:**
   - Right-click and save images
   - Follow attribution requirements (if any)
   - Check licenses (commercial use allowed)

2. **Optimize Images:**
   - Use online tools: TinyPNG, ImageOptim, Squoosh
   - Target size: Product images ~100-200KB, Gallery ~200-300KB
   - Format: JPG for photos, WebP for best compression

3. **Rename Files:**
   - Use descriptive names: `automotive-harness-1.jpg`
   - Keep lowercase with hyphens
   - Match the names in HTML if replacing placeholders

4. **Add to Website:**
   - Place in appropriate `/images` subfolder
   - Replace placeholder image paths in HTML
   - Test loading on slow connections

## Image Specifications

### Recommended Sizes:
- **Hero Images:** 1920x1080px or larger
- **Product Images:** 800x600px minimum
- **Gallery Images:** 1200x800px or 600x400px
- **Thumbnails:** 400x300px

### File Formats:
- **JPG:** Best for photographs (80-90% quality)
- **WebP:** Modern format, best compression
- **PNG:** For images with transparency (not needed here)

### Optimization:
- **Target File Size:** 100-300KB per image
- **Tools:**
  - Online: TinyPNG.com, Squoosh.app
  - Desktop: ImageOptim (Mac), RIOT (Windows)
